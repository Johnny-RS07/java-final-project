# java-final-project
This is for my CSCI project

This is a program that houses a bunch of small minigames that you can play.
These games include Rock Paper Scissors, Guess the number, Trivia, Higher or Lower, and Word Scramble.
When running this program you will be prompted with a menu of all the games that I have listed that you can chose from.
When playing Guess the Number you will be prompted to enter a number and the computer will tell you if you are to high or to low.
When play Rock Paper Scissors you will play the game Rock Paper Scissors agaisnt a robot.
When playing the trivia game you will be asked trivia questions where you will be prompted to chose one of three option and at the end it will tell you how many you got right.
When playing word scramble you will be presented with a buch of letters and you the user will be prompted to descramble the word prompted.
When playing Higher or lower you will be given a number from 1 - 13 and you will be prompted if the next number will be higher or lower of the current number.

the Game.java file is the super class for all the games.

The main file is were the user is put to actually run the program.

the Rock paper scissors file is were the rock paper scissors game is made and logic for the game is stored.

the guess the number file is were the guess the number game is made and logic for the game is stored.

the Trivia file is were the Trivia game is made and logic for the game is stored.

the Word Scramble file is were the Word Scramble game is made and logic for the game is stored.

the High low file is were the High low game is made and logic for the game is stored.

All you have to do is to enter a number correrlated to the game you want to play then follow the prompts given to you to finish the game.
